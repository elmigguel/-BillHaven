## ğŸ› Troubleshooting

### "Logos not loading"
- Check CDN links in VISUAL_ASSETS_CDN_LINKS.md
- Verify CORS settings
- Use local assets as fallback
- Check network tab for 404s

### "Colors look wrong"
- Verify Tailwind config has chain colors
- Check dark mode overrides
- Use official brand colors from guide
- Test in light and dark mode

### "Components look broken on mobile"
- Use mobile-first Tailwind classes
- Test on real devices (not just browser)
- Check touch target sizes (44px minimum)
- Use MobileBottomSheet for modals

### "TypeScript errors"
- Install @types packages
- Check type definitions in components
- Use `ChainId` and `Token` types
- Enable strict mode gradually
